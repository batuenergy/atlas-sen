# Security & data-provenance policy

## Reporting a vulnerability
Email **contacto@batuenergy.com** with details. Please don't open a public issue for
security problems. We aim to acknowledge within a few business days.

## Data provenance reports
This project is careful about data licensing. If you believe a committed file contains
data that isn't compatible with `DATA-LICENSE.md` — for example, coordinates or geometry
from a proprietary source — please report it the same way. Provenance issues are treated
with the same priority as security issues.
